Ajuda

Encontre aqui algumas informações úteis sobre o aplicativo, guia de uso e contato.

Perguntas frequentes

Como deve ser a nomeação dos arquivos?

Para utilizar as ferramentas de convenção de nomeação, os nomes dos arquivos das imagens devem possuir alguma das estruturas, dividido por "-" ou "_".

Onde encontro o guia e a documentação completa?

O guia pode ser encontrado na seção abaixo, a documentação está disponível no repositório do aplicativo no github;

Como reporto um problema?

Para reportar um problema, abra uma issue no repositório do GitHub ou entre em contato através do email fornecido na seção de Contato.

Como faço para atualizar o aplicativo?

Para atualizar o aplicativo, acesse a seção de Atualizar no menu superior e siga as instruções, caso seu aplicativo se encontrar desatualizado.

Como faço para alterar as configurações?

Para alterar as configurações, acesse a seção de Configurações no menu superior e siga as instruções.

Como faço para utilizar o aplicativo?

Para utilizar o aplicativo, acesse a seção de Início no menu superior, selecione qual parte da planta deseja analisar, adicione as imagens e os comandos que deseja processar e o formato de saída desejável, para mais informações consulte o guia de uso.

O que é o modo de máxima performance?

Este modo força o uso de processamentos paralelos e de multiplos núcleos do CPU, levando a consideraveis ganhos de performance, porém com grande custo de recursos, essa opção pode afetar a estabilidade do aplicativo em alguns computadores, não é recomendável seu uso em máquinas fracas, para maiores informações consulte o guia de uso.
